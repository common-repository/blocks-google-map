Integrate Google Map to your post using Gutenberg.

== Description ==
Is your website is Gutenberg supported and you looking for block to integrate the maps into your post? If yes, To integreate the Google Maps you can use this plugin.

[youtube https://www.youtube.com/watch?v=0KUTj_N1ES8]

This plugin's code is available on [Github](https://github.com/emgk/Blocks-Google-Maps). Please feel free to fork the [repository](https://github.com/emgk/Blocks-Google-Maps) and send a pull request. If you find a bug in the plugin, open an issue.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/Blocks-Google-Map` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= Does this plugin requires Google Map API key? =
Yes, it does. Go to WP-Admin > "Google Map API".
